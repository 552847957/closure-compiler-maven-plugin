Is it possible to create multiple CSS & JavaScript bundles?

Yes, it is. Just use multiple <execution> tags with different final names.

Note that execution id's have to be unique among all executions of a single plugin within a POM.

[top]

Where can I find the complete list of configuration parameters?

See minify:minify goal for more info.

[top]

Is it possible to use a different JavaScript compiler?

Yes, Google Closure Compiler can be configured using the option <jsEngine>closure</jsEngine>.
