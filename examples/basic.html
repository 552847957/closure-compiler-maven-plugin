Basic Configuration

Configure Minify Maven Plugin

This is the most basic configuration to run Minify Maven Plugin everytime the project is built.

<project>
  <!-- ... -->
  <build>
    <plugins>
      <!-- ... -->
      <plugin>
        <groupId>com.samaxes.maven</groupId>
        <artifactId>minify-maven-plugin</artifactId>
        <version>1.4</version>
        <configuration>
          <cssSourceFiles>
            <cssSourceFile>file-1.css</cssSourceFile>
            <!-- ... -->
            <cssSourceFile>file-n.css</cssSourceFile>
          </cssSourceFiles>
          <jsSourceFiles>
            <jsSourceFile>file-1.js</jsSourceFile>
            <!-- ... -->
            <jsSourceFile>file-n.js</jsSourceFile>
          </jsSourceFiles>
        </configuration>
      </plugin>
      <!-- ... -->
    </plugins>
  </build>
  <!-- ... -->
</project>

Invoke from the command-line

Since the minify goal is not bound to any phase in the build cycle, you need to explicitly invoke minify:minify goal to create a package containing the new files.

mvn minify:minify package
