minify:minify

Full name:

com.samaxes.maven:minify-maven-plugin:1.7:minify

Description:

Goal for combining and minifying CSS and JavaScript files.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-resources.

Optional Parameters

Name Type Since Description
bufferSize int - Size of the buffer used to read source files.
Default value is: 4096.
charset String 1.3.2

If a supported character set is specified, it will be used to read the input file. Otherwise, it will assume that the platform's default character set is being used. The output file is encoded using the same character set.

See the IANA Charset Registry for a list of valid encoding types.


Default value is: UTF-8.
cssFinalFile String - CSS output filename.
Default value is: style.css.
cssSourceDir String - CSS source directory.
Default value is: css.
cssSourceExcludes ArrayList 1.2 CSS files to exclude. Specified as fileset patterns which are relative to the CSS source directory.
cssSourceFiles ArrayList - CSS source filenames list.
cssSourceIncludes ArrayList 1.2 CSS files to include. Specified as fileset patterns which are relative to the CSS source directory.
cssTargetDir String 1.3.2 CSS target directory.

Takes the same value as cssSourceDir when empty.
debug boolean 1.5.2 Show source file paths in log output.
Default value is: false.
disableOptimizations boolean - JAVASCRIPT ONLY OPTION!
Disable all the built-in micro optimizations.
Default value is: false.
jsEngine String 1.6

JAVASCRIPT ONLY OPTION!
Define the JavaScript compressor engine to use.

Possible values are:

  • yui - YUI Compressor
  • closure - Google Closure Compiler

Default value is: yui.
jsFinalFile String - JavaScript output filename.
Default value is: script.js.
jsSourceDir String - JavaScript source directory.
Default value is: js.
jsSourceExcludes ArrayList 1.2 JavaScript files to exclude. Specified as fileset patterns which are relative to the JavaScript source directory.
jsSourceFiles ArrayList - JavaScript source filenames list.
jsSourceIncludes ArrayList 1.2 JavaScript files to include. Specified as fileset patterns which are relative to the JavaScript source directory.
jsTargetDir String 1.3.2 JavaScript target directory.

Takes the same value as jsSourceDir when empty.
linebreak int - Some source control tools don't like files containing lines longer than, say 8000 characters. The linebreak option is used in that case to split long lines after a specific column. It can also be used to make the code more readable, easier to debug (especially with the MS Script Debugger). Specify 0 to get a line break after each semi-colon in JavaScript, and after each rule in CSS. Specify -1 to disallow line breaks.
Default value is: -1.
nomunge boolean - JAVASCRIPT ONLY OPTION!
Minify only. Do not obfuscate local symbols.
Default value is: false.
nosuffix boolean 1.7 Do not append a suffix to the minified output filename, independently of the value in the suffix parameter.
Default value is: false.
preserveAllSemiColons boolean - JAVASCRIPT ONLY OPTION!
Preserve unnecessary semicolons (such as right before a '}'). This option is useful when compressed code has to be run through JSLint (which is the case of YUI for example).
Default value is: false.
skipMerge boolean 1.5.2 Skip the merge step. Minification will be applied to each source file individually.
Default value is: false.
skipMinify boolean 1.5.2 Skip the minify step. Useful when merging files that are already minified.
Default value is: false.
suffix String 1.3.2 The output filename suffix.
Default value is: min.
verbose boolean - JAVASCRIPT ONLY OPTION!
Display informational messages and warnings.
Default value is: false.
webappSourceDir String - Webapp source directory.
Default value is: ${basedir}/src/main/webapp.
webappTargetDir String - Webapp target directory.
Default value is: ${project.build.directory}/${project.build.finalName}.
